## 1. 보드 개요

보드는 프로젝트 내에서 시각적 레퍼런스를 수집하고 정리할 수 있는 무한 캔버스입니다.

### 주요 특징
- **무한 캔버스**: 줌/팬으로 자유롭게 이동
- **다양한 요소**: 이미지, 영상, 텍스트, 도형, 스티키 노트, 프레임 지원
- **실시간 저장**: 변경 사항 자동 저장
- **공유 기능**: 링크로 보드 공유 가능

---

## 2. 보드 생성 및 관리

### 보드 생성
1. 프로젝트 상세 페이지에서 "보드" 탭 클릭
2. "새 보드 만들기" 버튼 클릭
3. 보드 제목 입력 후 생성

### 보드 관리
- **제목 변경**: 헤더의 제목 클릭 → 수정 → Enter
- **삭제**: 헤더 우측 ⋮ 메뉴 → "보드 삭제"
- **공유**: "공유" 버튼 클릭 → 링크 복사

---

## 3. 캔버스 조작

### 줌 (확대/축소)
| 동작 | 방법 |
|------|------|
| 확대 | `Ctrl` + 마우스 휠 위 또는 `Ctrl` + `+` |
| 축소 | `Ctrl` + 마우스 휠 아래 또는 `Ctrl` + `-` |
| 100% | `Ctrl` + `0` |
| 줌 프리셋 | 하단 줌 컨트롤의 퍼센트 클릭 → 선택 |

### 팬 (이동)
| 동작 | 방법 |
|------|------|
| 상하좌우 이동 | 마우스 휠 (Ctrl 없이) |
| 드래그 이동 | 빈 캔버스 드래그 (선택 도구 모드) |

### 그리드 표시
- 헤더의 그리드 아이콘 클릭으로 토글
- 요소 배치 시 정렬에 도움

## 4. 도구 모음

좌측 도구 모음에서 사용할 도구를 선택합니다.

| 아이콘 | 도구 | 단축키 | 설명 |
|--------|------|--------|------|
| 화살표 | 선택 | `V` | 요소 선택/이동/크기조절 |
| T | 텍스트 | `T` | 텍스트 박스 생성 |
| 사각형 | 도형 | - | 다양한 도형 생성 (드롭다운) |
| 메모 | 스티키 노트 | `N` | 메모 생성 |
| 이미지 | 이미지 | - | 이미지 파일 추가 |
| 영상 | 영상 | - | 영상 파일 추가 |
| 프레임 | 프레임 | `F` | 프레임 템플릿 생성 (드롭다운) |

## 5. 요소 생성

### 5.1 텍스트

1. 도구 모음에서 텍스트(T) 선택 또는 `T` 키
2. 캔버스 클릭
3. 더블클릭하여 텍스트 편집 또는 우측 속성 패널에서 입력

### 5.2 도형

도구 모음의 도형 드롭다운에서 선택:

| 도형 | 설명 |
|------|------|
| 사각형 | 모서리 둥글기 조절 가능 |
| 원 | 정원형 |
| 삼각형 | 정삼각형 |
| 화살표 | 방향 표시 |
| 별 | 5각 별 |
| 선 | 직선 |

### 5.3 스티키 노트

1. 도구 모음에서 스티키 노트 선택 또는 `N` 키
2. 캔버스 클릭하여 생성
3. 6가지 색상 중 선택 가능 (속성 패널)

### 5.4 프레임 템플릿

도구 모음의 프레임 드롭다운에서 선택:

| 템플릿 | 크기 |
|--------|------|
| 사용자 정의 | 400×300 |
| Phone | 375×667 |
| Phone Pro | 390×844 |
| Tablet | 768×1024 |
| Desktop | 1440×900 |
| Desktop HD | 1920×1080 |
| A4 | 595×842 |
| Instagram Post | 1080×1080 |
| Instagram Story | 1080×1920 |
| YouTube Thumbnail | 1280×720 |

### 5.5 이미지/영상

1. 도구 모음에서 이미지 또는 영상 아이콘 클릭
2. 파일 선택 대화상자에서 파일 선택
3. 캔버스에 자동 배치

## 6. 요소 편집

### 선택
| 동작 | 방법 |
|------|------|
| 단일 선택 | 요소 클릭 |
| 다중 선택 (추가) | `Shift` 또는 `Ctrl` + 클릭 |
| 다중 선택 (영역) | 빈 캔버스에서 드래그 |
| 전체 선택 | `Ctrl` + `A` |
| 선택 해제 | `Esc` 또는 빈 캔버스 클릭 |

### 이동
- 선택 후 드래그
- 속성 패널에서 X, Y 좌표 직접 입력

### 크기 조절
- 선택 후 핸들(모서리/가장자리) 드래그
- 속성 패널에서 W, H 직접 입력

### 회전
- 선택 후 회전 핸들 드래그
- 속성 패널에서 각도 직접 입력

### 복제
- `Ctrl` + `D` 또는 속성 패널의 복사 버튼
- 선택한 요소가 약간 이동된 위치에 복제됨

### 삭제
- `Delete` 또는 `Backspace` 키
- 속성 패널의 휴지통 버튼

## 8. 속성 패널

우측 패널에서 선택된 요소의 속성을 편집합니다.

### 공통 속성
| 속성 | 설명 |
|------|------|
| 위치 (X, Y) | 캔버스 내 좌표 |
| 크기 (W, H) | 너비, 높이 |
| 회전 | 각도 (0~360°) |
| 투명도 | 0% ~ 100% |
| 잠금 | 이동/편집 방지 |

### 텍스트/스티키 노트 속성
| 속성 | 설명 |
|------|------|
| 텍스트 내용 | 직접 입력 가능 |
| 폰트 크기 | 12px ~ 72px |
| 글자색 | 컬러 피커 |
| 정렬 | 왼쪽/가운데/오른쪽 |
| 굵게/기울임 | 토글 버튼 |

### 도형 속성
| 속성 | 설명 |
|------|------|
| 배경색 | 채우기 색상 |
| 테두리색 | 선 색상 |
| 테두리 두께 | 0 ~ 10px |
| 모서리 둥글기 | 0 ~ 50 (사각형만) |

### 스티키 노트 색상
6가지 프리셋 색상:
- 노랑, 초록, 파랑, 분홍, 주황, 보라

### 레이어 순서
| 버튼 | 기능 |
|------|------|
| ⬆⬆ | 맨 앞으로 |
| ⬆ | 앞으로 |
| ⬇ | 뒤로 |
| ⬇⬇ | 맨 뒤로 |

### 정렬 (2개 이상 선택 시)
**수평 정렬**: 왼쪽, 가운데, 오른쪽
**수직 정렬**: 위, 가운데, 아래
**균등 분배**: 가로, 세로 (3개 이상 선택 시)

## 10. 내보내기

헤더 우측 ⋮ 메뉴에서 내보내기 옵션을 선택합니다.

### PNG 내보내기
- 현재 캔버스를 고해상도 PNG 이미지로 저장
- 파일명: `board-export-YYYY-MM-DD.png`

### JSON 내보내기
- 보드 구조를 JSON 형식으로 저장
- 백업 또는 데이터 분석 용도
- 파일명: `board-export-YYYY-MM-DD.json`

## 하단 컨트롤

캔버스 하단에 위치한 컨트롤 바입니다.

### 실행 취소/다시 실행
- 최대 50단계 히스토리 지원

### 정렬 도구 (2개 이상 선택 시)
- 요소들을 정렬하고 균등 분배

### 줌 컨트롤
- 슬라이더로 줌 조절
- 현재 줌 퍼센트 표시
- 프리셋 줌 선택 가능

---

## 팁 & 트릭

1. **빠른 복제**: 요소 선택 후 `Ctrl` + `D`로 빠르게 복제
2. **정확한 배치**: 그리드를 켜고 스냅 기능 활용
3. **다중 선택 후 정렬**: 여러 요소를 선택하고 정렬 버튼으로 깔끔하게 정리
4. **레이어 관리**: 많은 요소가 있을 때 레이어 패널에서 쉽게 찾기
5. **작업 실수 복구**: `Ctrl` + `Z`로 최대 50단계까지 되돌리기
